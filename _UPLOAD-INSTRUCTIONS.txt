Deen o Dunya Planner — publish on GitHub Pages (no coding)
==========================================================
You will upload these files to a new GitHub repo, then turn on Pages.

A) CREATE THE REPO
   1. Go to https://github.com/new
   2. Repository name:  deen-o-dunya   (lowercase, no spaces)
   3. Set it to PUBLIC. Do NOT add a README. Click "Create repository".

B) UPLOAD THESE FILES  (index.html must end up at the repo ROOT)
   1. On the new repo page click:  "uploading an existing file"
      (link is in the "...or upload an existing file" line)
   2. Unzip this package first, then SELECT ALL the items INSIDE it
      (index.html, manifest.webmanifest, sw.js, .nojekyll, the js/ vendor/
       icons/ folders) and drag them into the browser.
   3. Wait for them to finish uploading, then click "Commit changes".

C) TURN ON PAGES
   1. In the repo go to:  Settings -> Pages
   2. Under "Build and deployment" > Source, choose "Deploy from a branch".
   3. Branch: main   Folder: / (root).  Click Save.
   4. Wait ~1 minute, refresh. It will show your live URL:
      https://YOUR-USERNAME.github.io/deen-o-dunya/

D) SEND ME THAT URL.  That's it.

Tip: open the URL on your Android phone, Chrome menu -> "Add to Home screen"
to see it install like a real app.
